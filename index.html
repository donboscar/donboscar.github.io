Diseño de una página web

Oscar Hubble

Digamos que quieres construir una página web...

Qué es lo primero que tienes que hacer?

~ 1º ~

Lo primero y más importante es pensar en la función y el contenido de la web. Solo después podemos empezar a darle vida y forma.

Vamos a asumir que ya has pensado en el contenido de tu página web, lo primero que tendrás que hacer es crear un documeto style.css. ¿Qué es lo primero que podemos diseñar?

Centrar

Queda muy feo ver todo el contenido pegado a la izquierda de la pestaña . Lo arreglamos centrando el contenido y limitando el número de caracteres por linea.

body {
    margin: 0 auto;
    max-width: 50em;
}

Una vez formateado el posicionamiento del contenido es hora de formatear el propio texto.

Font

Por defecto, el navegador usa "Times", si no es el estilo que estás buscando, puedes camabiar. Cambiar a sans-serif, "Helvetica" o "Arial". En este caso importamos tres en orden de preferencia, si el primero falla pasa al segundo y así sucesivamente. Por lo general se suele acabar en un estilo como "serif" que es compatible con todos los navegadores.

body {
    font-family: "Helvetica", "Arial", sans-serif;
}

Vale, ya hemos conseguido una bonita tipografía, toca hacerlo más legible.

Espacios e Interlineado

Los espacios de un documento tienen un impacto directo sobre su legibilidad. El espacio en márgenes e interlineado aumentan la comprensión y velocidad de lectura.

body {
      line-height: 1.5;
      padding: 4em 1em;
    }
    
    h2 {
      margin-top: 1em;
      padding-top: 1em;
    }

Hemos mejorado mucho el formato del documento, pero algo de color no haría daño.

Porque sí, porque es la moda

Es muy frecuente en la Web ver el contenido escrito en diferentes tonos de gris.

body {
      color: #555;
    }

Pero, para seguir manteniendo un buen nivel de contraste, escogemos un tono más oscuro en los encabezados y palabras importantes.

h1,
    h2,
    strong {
      color: #333;
    }

Hemos separado los fragmentos de código del resto del escrito con una tipografía diferente. Podemos ir un paso más allá.

Balance

Hemos añadido un fondo de contraste detrás de los fragmentos de código:

code,
    pre {
      background: #eee;
    }
    
    code {
      padding: 2px 4px;
      vertical-align: text-bottom;
    }
    
    pre {
      padding: 1em;
    }

Podemos perseguir un tema cromático uniforme por toda la Web.

Tema Cromático

La mayoría de empresas, como un partido político, tienen un color representativo que actúa como acento visual. En una página Web puede ser un color diferente en los links.

a {
      color: #e81c4f;
    }

Añadimos unos colores secundarios.

Colores Secundarios

La cromática de la Web ya está muy conseguida. Hemos añadido sombras sutiles en los bordes y en el fondo.

body {
      color: #566b78;
    }
    
    code,
    pre {
      background: #f5f7f9;
      border-bottom: 1px solid #d8dee9;
      color: #a7adba;
    }
    
    pre {
      border-left: 2px solid #69c;
    }

Ahora he decidido que quiero una tipografía diferente...

Roboto

Esta es una tipografía típica de Android.

También puedes incrustar tu propia letra o solicitar un servicio online como Typekit. En nuestro caso "Roboto" de Google Fonts service:

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';
    
    body {
      font-family: "Roboto", "Helvetica", "Arial", sans-serif;
    }

Suficiente formateo de texto, toca añadir mil palabras...

El apartado gráfico de una página Web es fundamental, no solo como ornamento sino pudiendo ser parte principal del mensaje a transmitir.

Por último, vamos a mejorar el encabezado con una imagen de fondo sacada de Unsplash

header {
      background-color: #263d36;
      background-image: url("header.jpg");
      background-position: center top;
      background-repeat: no-repeat;
      background-size: cover;
      line-height: 1.2;
      padding: 10vw 2em;
      text-align: center;
    }

Añadimos un logo

header img {
      display: inline-block;
      height: 120px;
      vertical-align: top;
      width: 120px;
    }

Cambiamos el encabezado a blanco y creamos un botón

header h1 {
      color: white;
      font-size: 2.5em;
      font-weight: 300;
    }
    
    header a {
      border: 1px solid #e81c4f;
      border-radius: 290486px;
      color: white;
      font-size: 0.6em;
      letter-spacing: 0.2em;
      padding: 1em 2em;
      text-transform: uppercase;
      text-decoration: none;
      transition: none 200ms ease-out;
      transition-property: color, background;
    }
    
    header a:hover {
      background:  #e81c4f;
      color: white;
    }

Y C'est fini!

Hemos creado una web "estática".
